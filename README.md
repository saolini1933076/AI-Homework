## N-Queens AI Homework
Implementation of the n-Queens problem using:

A* search (with duplicate elimination, no reopening)
CSP (OR-Tools CP-SAT solver)
 
 ## How to run
Install dependencies: pip install ortools
Execute everything with: python nqueens_ai.py
The results will be saved in results.csv with metrics for A* and CSP.
